Dashboard para Análise de Dados

    Este projeto é uma dashboard desenvolvida em para monitoramento e análise de dados financeiros, recursos humanos (RH) e logísticos. A aplicação permite aos usuários visualizar, de forma clara e interativa, os principais indicadores de custos dessas áreas, ajudando na tomada de decisões estratégicas.

  • Cards Interativos: 12 cards distribuídos em três categorias (Financeiro, RH, Logística), com informações como total de registros, soma dos gastos, média de gastos e maior valor registrado em cada área.
  • Filtros Personalizados: Possibilidade de filtrar os dados por ano, mês e período, permitindo uma análise detalhada e ajustada conforme a sua necessidade.
  • Gráficos Dinâmicos: Gráfico Houghnout para mostrar a distribuição dos valores de cada área, e gráfico Radar para visualizar a performance ao longo dos meses do ano, com linhas separadas para financeiro, RH e logística.
  • Gráfico de Barras por Estado: Um gráfico de barras localizado no lado direito da dashboard, que exibe os gastos totais financeiros, RH e logísticos por estado do Brasil. Esse gráfico facilita a comparação entre diferentes regiões e permite entender as variações de custos em cada localidade.
  • Filtros Específicos para Gráficos: Cada gráfico possui um filtro que permite ajustar os dados para cada categoria (financeiro, RH ou logística), garantindo uma análise detalhada e específica de cada área.
  • Desempenho e Escalabilidade: O sistema foi desenvolvido com foco em performance, utilizando o poder do Laravel 10 para garantir um backend rápido e responsivo, que lida com grandes volumes de dados de forma eficiente. A estrutura modular permite fácil expansão e adição de novas funcionalidades.
  • Arquitetura MVC: Todo o backend foi desenvolvido seguindo os padrões de MVC, com rotinas bem definidas para controle de dados (Model), lógica de negócios (Controller) e apresentação (View). Isso assegura que o sistema seja fácil de manter e evoluir ao longo do tempo.
  • Segurança e Autenticação: O sistema foi implementado com rigorosos controles de segurança, incluindo autenticação de usuários, validação de dados e permissões, utilizando as poderosas funcionalidades nativas do Laravel.

    Este projeto destaca minhas habilidades em análise de dados e desenvolvimento de dashboards interativas, utilizando gráficos avançados e filtros dinâmicos para apresentar informações complexas de maneira acessível e intuitiva.
