Gerenciamento de Agendamentos

    O Sistema de Gerenciamento de Agendamentos para Salão de Beleza desenvolvido em permite que clientes se cadastrem e agendem serviços, podendo editar seus agendamentos desde que faltem mais de dois dias para o atendimento.
    O proprietário do salão possui acesso administrador, com privilégios para cadastrar, editar e excluir serviços oferecidos. Apenas o administrador tem permissão para excluir agendamentos, garantindo controle total sobre o gerenciamento dos serviços e agendamentos no sistema.

  • Login e Registro:
    • Tela de Login: O administrador (Leila) acessa o painel administrativo utilizando o e-mail leila@gmail.com e a senha Gestao@2024.
    • Tela de Registro: Destinada aos clientes do salão, onde podem registrar seus agendamentos para serviços de beleza.
  • Agendamentos de Clientes:
    • Cadastro de Agendamentos: O cliente pode agendar serviços no salão. Se o cliente já tiver um agendamento para a mesma semana, o sistema oferecerá a opção de editar o agendamento existente ou cadastrar um novo no mesmo horário/data.
    • Edição de Agendamentos: O cliente só poderá editar os agendamentos se faltarem mais de dois dias para o atendimento. Caso contrário, a edição não será permitida.
  • Acesso do Administrador:
    • Tela Admin (Leila): O acesso do administrador leva à tela de dashboard, onde é possível visualizar os agendamentos, dados financeiros do salão e outras informações do negócio.
    • Cadastro de Serviços: O administrador pode cadastrar serviços oferecidos pelo salão, incluindo preços e descrição.
  • Estrutura de Diretórios:
    • app/: Contém os arquivos principais do seu aplicativo, incluindo controladores, modelos e serviços.
    • database/: Contém migrações, seeds e fábricas.
    • resources/views/: Contém as views Blade do sistema.
    • routes/: Contém as rotas da aplicação.
  • Segurança e Autenticação: O sistema foi implementado com rigorosos controles de segurança, incluindo autenticação de usuários, validação de dados e permissões, utilizando as poderosas funcionalidades nativas do Laravel.
  • Documentação Completa: O sistema está completamente documentado no arquivo README, com todas as instruções necessárias para rodar o projeto. A documentação inclui orientações detalhadas para executar as migrations e seeders, garantindo que o ambiente de desenvolvimento esteja corretamente configurado.

    O sistema foi projetado para ser fácil de usar, com ênfase em uma navegação simples e funcionalidades bem definidas, garantindo uma experiência agradável tanto para os visitantes quanto para os usuários registrados.
